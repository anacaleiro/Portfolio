Ana Pedro Caleiro Portfolio

The Workersâ€™ Compensation Board (WCB) processes thousands of claims annually, requiring a scalable approach to classify claim injury types efficiently. This project aimed to automate the classification process using supervised machine learning models trained on over 593 thousand claims from 2020 to 2022.

Through extensive data preprocessing, feature engineering, and model evaluation, we tested multiple classification algorithms, including XGBoost, Random Forest, and LightGBM, with Stratified Cross-Validation to ensure reliability. XGBoost emerged as the top-performing model, achieving an F1-score of 0.47, Precision of 0.55, and Recall of 0.45. Given the imbalanced nature of the target variable, F1-Score (macro) was prioritized. The project also incorporated an interactive Streamlit-based interface to provide real-time claim classification insights.
